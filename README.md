# Virtual Image Drag & Drop Using OpenCV

## Description

This project uses OpenCV and MediaPipe Hand Tracking to move an object on the screen using finger gestures.

## Technologies

- Python
- OpenCV
- cvzone
- MediaPipe

## Features

- Real-time hand tracking
- Gesture detection
- Drag & Drop
- Low latency

## Run

```bash
pip install -r requirements.txt
python app.py
```
